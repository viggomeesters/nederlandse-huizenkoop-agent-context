# Agent Instructions

This is a public Dutch Markdown-first context repository. Keep it readable for humans and copy-pasteable for AI agents.

Rules:

- Write public content in Dutch and preserve Dutch home-buying terminology.
- Use only public, official, or clearly attributed sources.
- Never add real buyer dossiers, live bids, addresses, private letters, income data, mortgage documents, screenshots, or personal details.
- Use synthetic examples only.
- Every content page in `docs/` must include a `## Bronnen` section.
- Run `make check` before committing.
