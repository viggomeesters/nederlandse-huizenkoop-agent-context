# Repo-complete status

Publieke gereedheid vereist:

- README met doel, gebruik, waarschuwingen en route;
- license, security, contributing, support, changelog en code of conduct;
- synthetische voorbeelden;
- lokale validatie via `make check`;
- publieke repo-hero;
- geen echte adressen, biedingen, persoonsgegevens, hypotheekdocumenten of privécommunicatie.

Run:

```bash
make check
```
