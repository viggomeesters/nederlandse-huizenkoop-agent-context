# Contributing

Contributions are welcome if they keep the repository Dutch, source-backed and buyer-first.

Requirements:

- Use Dutch terminology.
- Add sources for factual claims.
- Use synthetic examples only.
- Do not add private cases or live bid data.
- Run `make check` before opening a pull request.

Useful contribution types:

- better source links;
- clearer checklists;
- additional synthetic examples;
- corrected legal/financial wording;
- better prompts for AI agents.
